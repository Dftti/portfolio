Systems That Learn

Most automations run the same way forever. Mine get better every cycle.

The Autoresearch Loop

Inspired by Karpathy's autoresearch framework. Every action the system takes is a hypothesis-driven experiment. After execution, it automatically measures results, interprets them with AI, and feeds confirmed learnings back into the next cycle.

Generate AI creates content with an embedded hypothesis to test
Execute Human approves, system sends via production APIs
Measure Auto-fetches real metrics after a waiting period
Learn AI interprets results — confirm, refute, or inconclusive
Improve Confirmed learnings feed into next generation prompt

    A["🎯 Hypothesis\nLibrary"] --> B["🤖 AI Generates\nwith experiment"]
